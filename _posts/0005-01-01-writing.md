<section markdown="1">
# Everything is a Module
The directory of your project is a module.

Modules commonly have a file for main, outputs, & variables.
{% highlight sh %}
minimal-terraform/
├── main.tf
├── outputs.tf
└── variables.tf
{% endhighlight %}
</section>


<section markdown="1">
# Modules are Modules

> It's modules all the way down.
{% highlight sh %}
my-project/modules/
└── aws
    ├── bastion
    │   ├── main.tf
    │   ├── outputs.tf
    │   ├── security_group.tf
    │   └── variables.tf
    └── stardog
        ├── main.tf
        ├── outputs.tf
        └── variables.tf
{% endhighlight %}
</section>

<section markdown="1">
# Grab Modules from _Places_
Modules are usually included in main.tf

{% highlight json %}
module "foo" {
  source = "./modules/bar/foo"
}

module "foo" {
  source = "git@github.com:hashicorp/example.git//foo"

}

module "consul" {
  source = "hashicorp/consul/aws"
}
{% endhighlight %}
